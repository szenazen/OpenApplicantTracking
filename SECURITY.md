# Security Policy

## Supported Versions

| Version | Supported |
| ------- | --------- |
| `main`  | Yes       |

## Reporting a Vulnerability

Please **do not** open a public issue for security vulnerabilities.

Email `security@openapplicanttracking.org` with:

- a description of the issue
- steps to reproduce
- affected versions / commits
- any known mitigation

We aim to acknowledge within 72 hours and provide a fix or mitigation within 14 days for critical issues.
